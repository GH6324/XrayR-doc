# 关于XrayR

## XrayR

A Xray backend framework that can easily support many panels.

一个基于Xray的后端框架，支持V2ay,Trojan,Shadowsocks协议，极易扩展，支持多面板对接。如果您喜欢本项目，可以右上角点个star+watch，持续关注本项目的进展。

[项目地址](https://github.com/XrayR-project)
